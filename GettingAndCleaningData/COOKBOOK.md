UCI HAR CleanData.txt
=================

This data  is analysis result from run_analysis.R against https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip, which was collected from the accelerometers from the Samsung GalaxyS smartphone. A full description is available at the site where the data was obtained:
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones


This data set has indexes of (SubjectId,Activity).For each index,it calculates average of mean and stdv ariables, please see below for more explanation.

SubjectId
This is id for each volunteer, that carried out the experiment. In total there are 30 volunteers within an age bracket of 19-48 years.

Activity
Each person performed six activities:(WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING)

tBodyAccmeanX
Average value of tBodyAcc-X per SubjectId and Activity. Same applies to all variables below. As to what these variables mean, please refer to its raw data definition in https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip.
